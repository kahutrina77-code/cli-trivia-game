# Trivia CLI Game

## Description
This project is a Command-Line Interface (CLI) trivia game built with JavaScript and Node.js. The game allows users to answer quiz questions, receive immediate feedback, track their score, and play within a time limit.

## Features
- Interactive CLI menu
- Sequential trivia questions
- Immediate feedback for answers
- Timed quiz session
- Final score summary

## Technologies Used
- JavaScript (ES Modules)
- Node.js
- @inquirer/prompts
- chalk
